Project Overview
In modern businesses, having one concentrated digital home base for products is extremely crucial as it connects distanced users to a singular product. In an introductory course, “Programming Usable Interfaces” at Carnegie Mellon, I mastered the basics of web development, gaining the flexibility to bring creative ideas to life and the technical insight needed to navigate websites effectively.

Using HTML, CSS, and JavaScript, I was able to showcase a BunBun Bake Shop’s unique offerings and create an engaging online presence. Starting from a blank slate allowed me to fully control the layout and functionality, ensuring the website was visually appealing, user-friendly, and responsive across devices. I implemented features like a gallery of baked goods, individual product galleries, and an interactive cart - all designed to enhance user engagement.

This project emphasized the importance of having a foundational understanding of web development, as it provides the ability to create designs into reality without relying on pre-built templates or external tools.

Product Gallery
Creating the product gallery for the BunBun Bakeshop website was a very detailed process that blended functionality with design. I began by structuring the gallery layout using HTML, ensuring each product was displayed within a responsive grid for optimal viewing on all devices. I styled the gallery with CSS, adding hover effects to highlight individual items and creating a cohesive look with color schemes and typography that reflected the bakeshop's branding. Through this container, row, column, and column layout, I was able to evenly align the different type of desserts offered from apple to strawberry cinnamon rolls.
To enhance interactivity, I incorporated JavaScript to enable features like efficient product displays, add to cart functions, and dynamic product descriptions. This process involved testing the gallery across different screen sizes and browsers to ensure a seamless user experience. The result was a visually engaging and easy-to-navigate showcase that not only highlighted the baked goods but also encourages customers to explore more, which could ultimately drive engagement and potential sales.
Add-to-Cart
Creating the "Add to Cart" feature for the BunBun Bakeshop website involved integrating HTML, CSS, and JavaScript to deliver a seamless user experience. I started by designing a visually appealing product display using HTML to structure the content and CSS to style it with clean, responsive layouts. Each product card featured an "Add to Cart" button, which I linked to JavaScript functionality.
Using JavaScript, I wrote code to dynamically track user selections, updating a virtual cart in real-time without requiring a page reload. This included displaying the selected items, their quantities, and the total cost in a cart. I also implemented logic to handle edge cases, such as removing items from the cart or adjusting quantities. By combining these technologies, I created an intuitive, efficient shopping experience that mirrors the interactivity users expect from modern e-commerce platforms. This process underscored the importance of blending design and functionality to meet user needs effectively.
Design & Functionality
Design
For the design of the BunBun Bakeshop website, I chose a warm tan color scheme complemented by yellow highlights and bold fonts to create a welcoming, appetizing atmosphere that reflects the cozy vibe of a neighborhood bakery. The tan background evokes feelings of comfort and pairs well with the theme of freshly baked goods, while the yellow highlights add a sense of energy and draw attention to key elements like call-to-action buttons and certain active states. For example, when the user is on the product or the cart HTML pages, they will be able to clearly know which page they are on and what pages are available to be clicked.
To enhance usability, I used bold fonts to differentiate active states, such as when users hover over the products , ensuring clear visual feedback. This approach not only reinforces the shop’s branding but also improves navigation by making important actions stand out. The combination of a cohesive color palette and intentional typography creates a user-friendly interface that is both aesthetically pleasing and functionally effective, encouraging users to engage with the site.
Functionality
The functionality of this website revolved around 3 main crucial points in order to create a seamless user experience:
When users click the "Add to Cart" button on a product, the number in the cart icon at the top right of the page dynamically updates to reflect the total items in the cart. JavaScript handles this logic by tracking each addition and ensuring the count remains accurate in real-time.
As users click into each different product page, the items each change dynamically based on one HTML page due to JavaScript. The prices, titles, and pictures of each Cinnamon Roll changes based on the user's choices
I implemented fully functional "Add" and "Remove" buttons within the cart interface, allowing users to adjust quantities or remove items altogether, with the total and cart number updating accordingly. This responsive design ensures users have complete control over their selections while maintaining an intuitive and error-free shopping experience, reinforcing the website's usability and reliability.

Reflection
Working on the BunBun Bakeshop website was an incredibly eye-opening experience that allowed me to sharpen my web development skills while exploring the creative and technical aspects of building a fully functional e-commerce platform. From designing a visually cohesive interface with a warm tan color scheme to coding essential features like the "Add to Cart" and “Remove” functionalities, this project challenged me to think holistically about user experience and technical execution. It taught me the importance of balancing aesthetics with usability, ensuring that every design choice enhances the site's functionality.
Additionally, creating dynamic interactions with JavaScript deepened my understanding of how front-end and back-end logic work together to deliver seamless user experiences. Reflecting on this project, I feel a strong sense of accomplishment in building something from scratch and gaining a deeper appreciation for the intricacies of modern web development. It reinforced my belief that mastering the basics is crucial for tackling more complex challenges in the future.
